# CRM Dashboard UI

##  Project Overview
This project is a simple CRM (Customer Relationship Management) dashboard UI designed using Figma.

##  Features
- Sidebar navigation (Dashboard, Leads, Clients, Tasks)
- Lead pipeline (New, Contacted, Proposal, Closed)
- Client list table
- Task / Follow-up tracker
- Dashboard summary cards

##  Tool Used
- Figma
